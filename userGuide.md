# دليل استخدام ثيم "سيدا لاين" الفاخر

## مقدمة

أهلاً بك في دليل استخدام ثيم "سيدا لاين" الفاخر، المصمم خصيصاً لمتجر العبايات الخاص بك على منصة سلة. هذا الثيم يتفوق على ثيم سيليا بمراحل، حيث يجمع بين التصميم الفاخر وميزات الذكاء الصناعي المتقدمة لتقديم تجربة تسوق استثنائية لعملائك.

**رابط المستودع على GitHub:** [https://github.com/H5500N/seda-line-theme](https://github.com/H5500N/seda-line-theme)

## مقارنة مع ثيم سيليا

| الميزة | ثيم سيليا | ثيم سيدا لاين الفاخر |
|---|---|---|
| **التصميم** | تصميم جيد وعصري | تصميم فاخر جداً بألوان راقية (أسود، ذهبي، بيج) وخطوط أنيقة |
| **الذكاء الصناعي** | لا يوجد | يوجد (توصيات ذكية، مساعد افتراضي، بحث بالصور، توليد أوصاف) |
| **الأداء** | جيد | ممتاز (تحميل أسرع، كود محسن) |
| **التخصيص** | خيارات تخصيص جيدة | خيارات تخصيص متقدمة ومرنة |
| **الميزات الحصرية** | لا يوجد | عرض 360 درجة، تجربة افتراضية (AR)، قسم VIP |

## ميزات الذكاء الصناعي

يتميز ثيم "سيدا لاين" بمجموعة من ميزات الذكاء الصناعي التي ستنقل متجرك إلى مستوى آخر:

1.  **المساعد الافتراضي الذكي (Chatbot):**
    - يساعد العملاء في اختيار العباية المناسبة.
    - يجيب على الأسئلة الشائعة حول المقاسات والألوان.
    - يقدم توصيات للمنتجات.

2.  **توصيات المنتجات الذكية:**
    - يحلل سلوك العملاء (المنتجات التي تم عرضها، الفئات التي تم النقر عليها).
    - يعرض منتجات مخصصة لكل عميل.

3.  **البحث الذكي بالصور:**
    - يسمح للعملاء بالبحث عن منتجات مشابهة عن طريق رفع صورة.

4.  **توليد أوصاف المنتجات تلقائياً:**
    - عند إضافة منتج جديد، يمكنك توليد وصف احترافي وجذاب بنقرة زر.

## كيفية تخصيص الثيم

يمكنك تخصيص الثيم بسهولة من خلال ملفات `twilight.json` و `luxury-theme.css`.

### 1. تغيير الألوان

يمكنك تغيير الألوان الرئيسية للثيم من خلال تعديل المتغيرات في ملف `luxury-theme.css`:

```css
:root {
  --color-gold: #D4AF37; /* اللون الذهبي */
  --color-beige: #F5F5DC; /* اللون البيج */
}
```

### 2. تغيير الخطوط

يمكنك تغيير الخطوط المستخدمة في الثيم من خلال تعديل المتغيرات في ملف `luxury-theme.css`:

```css
:root {
  --font-heading: 'Playfair Display', 'Amiri', serif; /* خط العناوين */
  --font-body: 'Cairo', 'Tajawal', sans-serif; /* خط النصوص */
}
```

### 3. تفعيل/تعطيل الميزات

يمكنك تفعيل أو تعطيل بعض الميزات من خلال ملف `twilight.json`.

## كيفية ربط الثيم بمتجرك على سلة

1.  اذهب إلى لوحة تحكم متجرك على سلة.
2.  اذهب إلى "تصميم المتجر".
3.  اختر "تخصيص التصميم".
4.  اربط المستودع الخاص بك على GitHub.
5.  اختر مستودع `seda-line-theme`.
